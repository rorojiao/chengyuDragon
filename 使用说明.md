# 成语接龙游戏 - 使用说明

## 快速启动

### 方法一：使用启动脚本（推荐）

```bash
cd "/Users/jiaojunze/Library/Mobile Documents/com~apple~CloudDocs/working_MAC/chengyuDragon"
./run_game.sh
```

这个脚本会自动：
1. 检查Python环境
2. 检查并安装依赖
3. 启动游戏

### 方法二：直接运行Python

```bash
cd "/Users/jiaojunze/Library/Mobile Documents/com~apple~CloudDocs/working_MAC/chengyuDragon"

```

## 如果遇到问题

### 问题1: 缺少依赖

如果提示缺少依赖包，运行：

```bash
/opt/homebrew/opt/python@3.13/bin/python3.13 -m pip install --user --break-system-packages PyQt6 PyYAML requests pypinyin httpx
```

### 问题2: 找不到Python

系统使用的是Python 3.13。如果路径不同，请修改run_game.sh中的PYTHON_EXE变量。

找到你的Python路径：
```bash
which python3
# 或
which python3.13
```

### 问题3: PyQt6导入失败

完全卸载并重新安装：
```bash
pip3 uninstall -y PyQt6 PyQt6-Qt6 PyQt6-sip
/opt/homebrew/opt/python@3.13/bin/python3.13 -m pip install --user --break-system-packages PyQt6
```

## 首次运行

1. **安装依赖**（自动）：
   ```bash
   ./run_game.sh
   ```
   首次运行会自动安装依赖

2. **配置LM Studio**（可选）：
   - 启动游戏
   - 点击"设置"
   - 配置API地址：`http://localhost:1234`
   - 点击"测试连接"

3. **开始游戏**：
   - 点击"开始游戏"
   - 输入你的成语
   - 与AI对战

## 游戏规则

- **接龙规则**：下一个成语的第一个字 = 上一个成语的最后一个字
- **不可重复**：已经使用过的成语不能再次使用
- **时间限制**：默认60秒（可设置）
- **提示功能**：有次数限制（可设置）
