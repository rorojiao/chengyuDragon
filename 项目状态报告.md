## 📋 使用说明

### 首次运行

1. **启动游戏**
   ```bash
   ./run_game.sh
   ```

2. **游戏模式**
   - **无需LM Studio**: 使用数据库模式，AI从数据库选择成语
   - **使用LM Studio**: 完整AI功能，需要配置API

3. **基本操作**
   - 点击"开始游戏"
   - 根据上一个成语的最后一个字输入你的成语
   - 点击"提交"或按回车
   - 与AI轮流接龙

### 配置LM Studio（可选）

1. 安装 [LM Studio](https://lmstudio.ai/)
2. 加载中文语言模型
3. 启动API服务器
4. 在游戏设置中配置：
   - API地址: `http://localhost:1234`
   - 点击"测试连接"
   - 保存设置

### 常见问题

```bash
/opt/homebrew/opt/python@3.13/bin/python3.13 -m pip install --user --break-system-packages PyQt6 PyYAML requests pypinyin httpx
```

**Q: 找不到Python？**
修改 `run_game.sh` 中的 `PYTHON_EXE` 为你的Python路径

**Q: 数据库找不到？**
确保 `resources/idioms.db` 存在

**Q: 游戏窗口无法显示？**
检查PyQt6是否正确安装
